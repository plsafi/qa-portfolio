## Project Description

This project presents the results of manual testing conducted on a demo e-commerce application. 
The main goal was to identify functional defects, usability issues, and inconsistencies across key user flows. 
Testing focused on areas such as the shopping cart, checkout process, product pages, and navigation. 
Various bugs were discovered, including critical issues affecting the payment process and cart functionality. 
The project demonstrates practical skills in exploratory testing, bug reporting, and creating structured test documentation.
